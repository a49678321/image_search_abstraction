FreeCodeCamp API Basejump: Image search abstraction

User stories:

  • Get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • Paginate through the responses by adding a /2 parameter to the URL.
  • Get a list of the most recently submitted search strings.

Example:

https://image-search-abstraction-hackerq.c9users.io/search/QUERY_as_you_want
